CNPPO Crime Reporting System

An E-Blotter system for the Camarines Norte Provincial Police Office.

One of my project when I was in college in 2013. A web based CRUD system we proposed to the PNP for faster,convenient and secure dissemination of information through its network of offices. The system also acts as an E-blotter system so as to replace the old fashioned pen and paper way.

Prints reports in PDF, secure user authentication, functional email server are some of it's features. Built using HTML, CSS, Javascript for front end. While the database and backend system were built with PHP Codeigniter framework and mySQL database.
